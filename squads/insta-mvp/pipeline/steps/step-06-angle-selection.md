# Step 06: Escolha do ângulo

## Contexto

Carla Carrossel gerou 5 ângulos em `squads/insta-mvp/output/{run_id}/angles.md`. O usuário escolhe um para o carrossel.

## Instruções

1. Ler angles.md do run_id atual.
2. Apresentar a lista numerada (label + hook_suggestion).
3. Pedir que responda com o **número** do ângulo escolhido (1 a 5).
4. Registrar o ângulo escolhido (label, hook_suggestion, emotional_driver) para o step 08 (Carla criar carrossel).

## Output

A escolha fica disponível para step-08. Runner segue para step-07 (tone selection).
