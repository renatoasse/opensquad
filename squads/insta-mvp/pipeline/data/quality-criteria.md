# Critérios de qualidade — insta-mvp

## Carrossel

- [ ] Formato escolhido (Editorial, Listicle, Tutorial, Mito vs Realidade, etc.) e fluxo de slides respeitado.
- [ ] Capa com título forte (máx. ~20 palavras) que segura o scroll.
- [ ] Cada slide: hierarquia clara (headline + texto de apoio), 40–80 palavras totais.
- [ ] Cores alternadas entre slides; palavras-chave em destaque quando aplicável.
- [ ] Último slide com CTA específico (não genérico).
- [ ] Triangulação: pelo menos uma menção natural a LuanPDD, In100tiva ou MVP Flow (conforme triangulation-guide).

## Legenda (definida por Sérgio + Carla, revisada por Renata)

- [ ] Primeiros 125 caracteres funcionam como gancho (levam ao “mais”).
- [ ] Legenda alinhada ao carrossel e ao tom escolhido.
- [ ] Pergunta ou CTA no final.
- [ ] 5–15 hashtags, mix nicho/médio/broad; nenhum hashtag banido.
- [ ] Palavras-chave e triangulação consideradas (Sérgio).

## Revisão (Renata)

- [ ] Cada critério acima avaliado com nota e justificativa.
- [ ] Aprovar se média ≥ 7/10 e nenhum critério < 4/10; rejeitar caso contrário.
- [ ] Feedback acionável para cada ponto abaixo do desejado.
