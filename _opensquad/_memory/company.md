## Identidade visual (MVP Flow / in100tiva)

Para peças visuais (carrosséis Instagram, arte, site): usar **sempre** a paleta do MVP Flow.

- **Preto:** fundo principal em temas escuros — `#000000` (ou `#0a0a0a`).
- **Branco:** texto principal em fundo escuro, botões em fundo roxo — `#FFFFFF`.
- **Roxo (acento principal):** destaque de frases, CTAs, botões, palavras-chave — `#8B5CF6` (ou `#7C3AED`); para variação mais clara/lavanda: `#A78BFA`.
- **Complementares:** cinza claro/off-white para texto secundário — `#E5E5E5`, `#A1A1AA`; fundo claro quando não for preto — `#FAFAFA` ou branco; grid ou overlay sutil — roxo muito escuro/desaturado `#1E1B2E` ou `#2D2A3E`.
- **Contraste:** manter texto legível (WCAG AA); em fundo preto usar branco ou cinza claro; em fundo claro usar preto ou cinza escuro; roxo para acentos e CTAs.
